


## Installation

```bash
$ ./dev.sh install_dependencies
```

## Running the app

```bash
# development
$ ./dev.sh start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e


```
